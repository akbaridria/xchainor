# x-chainor

## What is x-chainor ?
X-chainor is the bridge and swap that build on top of the axelar general message passing. that currently supported more than 4 chains.

## Demo App
  Try x-chainor [here](https://kaleidoscopic-fairy-9fb69f.netlify.app/).
  
## Dex Integration
- Arbitrum
  - Sushiswap
  - Curve
- BSC
  - Pancakeswap
  - Elipsis Finance
- Moonbeam
  - StellarSwap
- Polygon
  - QuickSwap
  - Curve
- Fantom
  - SpiritSwap
  - Curve
